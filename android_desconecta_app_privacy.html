English. Desconecta App Privacy Policy

  • This application does not store any personal private data.

  • The only data stored is the configuration of the application itself, i.e. when to activate/deactivate airplane mode.

  • The permissions needed are:
    • Phone
      • read phone status and identity
        • Permission needed to check if you have a phone call when it's time to activate Airplane Mode. If you're in a phone call, the app will wait 10 minutes and try again
    • Device ID & call information
      • read phone status and identity
        • Permission needed to check if you have a phone call when it's time to activate Airplane Mode. If you're in a phone call, the app will wait 10 minutes and try again
    • Other
      • run at startup
        • If you activate the "Alarm Airplane Mode" and you turn off the device, the application will create it again once the device is turned on.
      • prevent device from sleeping
        • Permission needed to use Android Alarm service.
      • modify system settings
        • Permission needed to activate Airplane Mode.

Spanish. Desconecta Política de Privacidad

  • Esta aplicación no almacena ninguna información privada del usuario.

  • Los únicos datos que almacena corresponden a la propia configuración de la aplicación, es decir, cuando se activa/desactiva el modo avión.

  • Los permisos necesarios son:
    • Teléfono
      • consultar la identidad y el estado del teléfono
        • Permiso necesario para comprobar si estamos hablando por teléfono antes de activar el Modo Avión, así evitamos finalizar la llamada.
    • ID de dispositivo y datos de llamada
      • consultar la identidad y el estado del teléfono
        • Permiso necesario para comprobar si estamos hablando por teléfono antes de activar el Modo Avión, así evitamos finalizar la llamada.
    • Otros
      • ejecutarse al inicio
        • Si hemos activado la aplicación y apagamos el dispositivo, al encenderlo de nuevo, se volverá a crear la alarma tal y como lo teníamos configurado.
      • impedir que el dispositivo entre en modo de suspensión
        • Permiso necesario para poder usar el sistema de alarmas de Android.
      • modificar los ajustes del sistema
        • Permiso necesario para activar el Modo Avión.
